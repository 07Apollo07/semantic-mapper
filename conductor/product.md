# Product Definition

Semantic Mapper AI is a tool designed to help users map source data (from PDFs, Excel) to target data models (like FSDM/ETL) using AI.

## Key Features
- **Knowledge Base Management**: Upload PDFs and Excel sheets to a vector store.
- **Project-based**: All data is organized into projects.
- **AI-assisted Mapping**: Use an agent to suggest mappings between source and target models.
- **Persistence**: Save project state, metadata, and mapping database.
