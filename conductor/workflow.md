# Workflow

1.  **Project Initialization**: Create or select a project.
2.  **Document Upload**: Upload PDFs or Excel files to the project's files directory.
3.  **Knowledge Base Sync**: Select specific Excel sheets or entire PDFs to index into the vector store.
4.  **Target Model Setup**: (Future) Define target tables and structures.
5.  **AI Mapping**: (Future) Run the agent to map source data to the target model.
6.  **Validation**: (Future) Review and validate AI-suggested mappings.
